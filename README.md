# DecodeLabs_Java_project-2
The Student Grade Calculator is a Java console-based application that helps evaluate student performance by calculating total marks, average percentage, and assigning grade accordingly.

FEATURES:
1.Accepts marks for any number of subjects.
2.Validates marks entered by the user.
3.Calculates total marks obtained.
4.Calculates average percentage.
5.Assigns grades based on average percentage.
6.Displays the maximum possible marks.
7.Console-based user interaction.

CONCEPTS USED:
1.Scanner Class
2.Loops
3.Conditional Statements
4.Input Validation
5.Arithmetic Operations

GRADE CRITERIA:
Average Percentage	Grade
90 and above	       A
80 - 89.99	         B
70 - 79.99	         C
60 - 69.99	         D
50 - 59.99	         E
Below 50	         Failed

HOW TO RUN THE PROJECT:

1.Clone the repository:
git clone <repository-link>

2.Navigate to the project directory.

3.Compile the Java file:
javac DecodeLabs_java_p2.java

4.Run the program:
java Calculator

5.Enter the number of subjects and marks when prompted.
